Fundraising Copilot Skill

Purpose

Help Solana founders prepare for fundraising by structuring narrative, metrics, pitch readiness, and investor expectations.

⸻

When to use

Use when user asks about:

* Raising capital
* Pitch decks
* Investor outreach
* Valuation strategy
* VC readiness
* Token/private sale planning

⸻

Input Required

* Project name
* Stage (pre-seed / seed / growth)
* Traction metrics (if any)
* Community size
* Revenue (if any)
* Token status

⸻

Fundraising Framework

1. Narrative Strength

Check:

* Is the story clear?
* Is there a strong problem statement?
* Is there ecosystem relevance (Solana alignment)?

⸻

2. Traction Metrics

Evaluate:

* Users
* Retention
* Growth rate
* Revenue (if any)

⸻

3. Market Position

Assess:

* Competitive advantage
* Timing advantage
* Ecosystem fit

⸻

4. Token Strategy (if applicable)

Check:

* Utility clarity
* Investor alignment
* Vesting fairness
* Inflation risk

⸻

Investor Readiness Score (0–100)

* Narrative
* Traction
* Market fit
* Token structure

⸻

Red Flags

* No clear product-market fit
* Weak retention metrics
* Overvalued early stage
* Poor token design

⸻

Output Format

Fundraising Readiness Score

⸻

Suggested Round Type

* Bootstrapping
* Angel round
* Seed round
* Not ready yet

⸻

Suggested Pitch Structure

1. Problem
2. Solution
3. Market
4. Traction
5. Token model
6. Use of funds

⸻

Action Plan

* What to fix before fundraising
* What to highlight
* What to avoid
