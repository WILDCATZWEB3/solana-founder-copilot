Partnership Strategy Skill

Purpose

Help Solana founders identify, structure, and execute ecosystem partnerships that improve distribution, credibility, and adoption.

⸻

When to use

Use when user asks about:

* Partnerships
* Ecosystem integrations
* BD strategy
* Collaborations
* Growth alliances
* Launch partners

⸻

Input Required

* Project type
* Stage
* Target users
* Current ecosystem position

⸻

Partnership Framework

1. Partnership Types

Infrastructure Partners

* RPC providers
* Wallets
* Indexers
* Data providers

Growth Partners

* Influencers
* KOLs
* Communities
* Launchpads

Ecosystem Partners

* Other Solana protocols
* DeFi integrations
* NFT platforms

⸻

2. Value Exchange Analysis

Evaluate:

* What you offer
* What you gain
* Alignment strength

⸻

3. Priority Ranking

Rank partnerships:

* Tier 1: Immediate impact
* Tier 2: Medium-term value
* Tier 3: Optional expansion

⸻

4. Outreach Strategy

Include:

* Cold DM templates
* Email structure
* Pitch framing

⸻

Output Format

Partnership Map

List top 5 recommended partnerships.

⸻

Priority Order

Ranked execution list.

⸻

Outreach Templates

Ready-to-send messages.

⸻

Expected Impact

* Growth
* Distribution
* Credibility
